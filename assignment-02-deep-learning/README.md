# Assignment 02 - Neural Networks and Deep Learning

This folder contains coursework from **CSE 574: Introduction to Machine Learning** focused on neural networks, deep learning, and image classification.

## Collaboration
Completed as a team coursework project.  
**My contribution:** Implemented **Part III** and **Part IV**, focusing on CNN-based image classification and a modified VGG-style architecture in PyTorch.

## Project Components

### 1. Tabular Neural Network
Built a feedforward neural network for binary classification on tabular data, including preprocessing, scaling, training, and evaluation.

**Notebook:** `notebooks/01_tabular_neural_network.ipynb`

### 2. Model Improvement Experiments
Explored techniques to improve neural network performance, including:
- batch normalization
- learning rate scheduling
- data augmentation
- gradient accumulation
- k-fold cross-validation

**Notebook:** `notebooks/02_model_improvement_experiments.ipynb`

### 3. CNN Image Classification
Developed a convolutional neural network for image classification using PyTorch, including dataset preparation, training, and evaluation.

**Notebook:** `notebooks/03_cnn_image_classification.ipynb`

### 4. Modified VGG-Style Image Classifier
Implemented and trained a modified VGG13-style architecture for multi-class image classification, with validation tracking and model checkpointing.

**Notebook:** `notebooks/04_modified_vgg_image_classifier.ipynb`
