# Intro to ML Coursework Projects

A curated collection of coursework projects completed as part of **CSE 574 - Introduction to Machine Learning** at the **University at Buffalo**. These assignments were cleaned, reorganized, and documented for portfolio presentation on GitHub.

## Included Projects

### Assignment 01 - Machine Learning Foundations
Topics covered:
- data preprocessing and exploratory data analysis
- logistic regression from scratch
- linear regression
- ridge regression
- elastic net regression
- model evaluation and visualization

### Assignment 02 - Neural Networks and Deep Learning
Topics covered:
- feedforward neural networks
- model improvement experiments
- convolutional neural networks (CNNs)
- modified VGG-style image classification
- PyTorch-based training and evaluation

## Tech Stack
Python, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, PyTorch, Jupyter Notebook
